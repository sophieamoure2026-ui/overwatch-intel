# OVERWATCH — Live Geopolitical Intelligence Terminal

Real-time military aircraft tracking, OSINT aggregation, and AI-powered threat analysis.

## Live Demo

Deployed at: [overwatch.onrender.com](https://overwatch.onrender.com)

## Features

- 🌍 Live ADS-B military aircraft map (247+ tracked)
- 📡 Real-time OSINT feed from 600+ sources
- 🤖 OVERWATCH AI analyst (Groq Llama)
- 📊 GeoRisk Score engine (0–100)
- ⚡ Automated trade directives
- 🔐 Subscription gating ($9–$29/mo)

## Stack

- Node.js + Express (server)
- Vanilla HTML/CSS/JS (frontend)
- ADS-B Exchange (aircraft data)
- Groq API (AI analyst)
- Render (hosting)

## Deploy

1. Fork this repo
2. Connect to Render as a Web Service
3. Build: `npm install` | Start: `node server.js`
4. Set `PORT` env var (Render does this automatically)
